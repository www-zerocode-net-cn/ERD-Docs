:::



## 前置条件

一台至少配置为 **4核** **8G** **可联网** 的Linux 主机。


## 一键安装

以 root 用户登录 Linux 主机执行以下命令：
``` bash
curl -SsL https://www.zerocode.net.cn/install | bash
```

安装完毕后，通过 **http://&lt;IP&gt;:9669** 即可访问 ERD Online 系统。
